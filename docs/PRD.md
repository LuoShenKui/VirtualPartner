# 二次元虚拟人物 Demo - PRD v2.0

## 🎯 项目目标

创建一个本地运行的二次元虚拟人物对话系统，支持：
- 自然语言对话（本地 LLM）
- 语音输入/输出
- 3D 角色展示（VRM）
- 表情/动作演出
- 记忆系统

## 🏗️ 技术架构

```
┌─────────────────────────────────────────────────────┐
│                    业务逻辑层                        │
│              (Rust 核心库 - 跨平台)                   │
│  - 对话系统  - 记忆管理  - 表情/动作选择  - VRM 解析  │
└─────────────────────────────────────────────────────┘
           │                        │
    ┌──────┴──────┐          ┌─────┴─────┐
    │   macOS     │          │  Windows  │
    │   Swift     │          │    C#     │
    │   AppKit    │          │  WPF/MAUI │
    └─────────────┘          └───────────┘
```

## 📋 MVP 功能

### 核心功能
- [ ] VRM 模型加载与显示
- [ ] 基础表情驱动（blendshape）
- [ ] 基础动画（idle/walk/sit）
- [ ] 对话系统（LLM + 记忆 + 心情）
- [ ] 简单场景（程序生成房间）

### 语音功能（Phase 3）
- [ ] Faster-Whisper 语音识别
- [ ] MeloTTS 语音合成
- [ ] 口型同步
